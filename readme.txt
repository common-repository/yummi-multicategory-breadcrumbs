Yummi Breadcrumbs is Super light weight & easy plugin with support SubCategories showing, SEO friendly and Auto Installation.

== Description ==

This plugin generates locational breadcrumb trails for your WordPress powered blog/website with support SubCategories showing and Auto Installation!

&emsp;<strong>Auto Installation!</strong> Now you not need put any code in files, just Install plugin and Activate it, that's all.

&emsp;<strong>SEO Friendly!</strong> Created based on SEO friendly 'Schema.org Microdata Breаdсrumbs' and 'RDFa Breadcrumbs' who so loves all search engines (Google, Bing, Yandex, etc...).

&emsp;<strong>Easy Settings!</strong> Settings can make even the most inexperienced user. User friendly Administrative interface makes setting options easy.

&emsp;<strong>Super Light Weight!</strong> Plugin Not using JavaScript and jQuery, pure and simple CSS, files contain minimal PHP code.

&emsp;Perfect <strong>worked with AMP (Accelerated Mobile Pages)!</strong>

&emsp;Perfect <strong>worked with taxonomy!</strong>

&emsp;<strong>ToolTips supported! Рure CSS.</strong>

== Installation ==

<strong>Auto Installation!</strong> Now you not need put any code in files, just Install plugin and Activate it, that's all.

for Manual Installation: go to Options > Yummi Breadcrumbs and switch there Installation to Manual

Open the appropriate file for your theme (typically header.php or single.php). This can be done within WordPress’ administration panel through Presentation > Theme Editor or through your favorite text editor. Place the following code where you want the breadcrumb trail to appear:
<pre><code><?php if(function_exists('yummi_breadcrumbs')) yummi_breadcrumbs(); ?></code></pre>
and this code to your AMP (Accelerated Mobile Pages) templates:
<pre><code><?php if(function_exists('yummi_breadcrumbs_amp')) yummi_breadcrumbs_amp(); ?></code></pre>

or Manual install from .zip:
<ol>
<li>Upload `yummi-breadcrumb` to the `/wp-content/plugins/` directory</li>
<li>Activate the plugin through the 'Plugins' menu in your blog/site</li>
<li>Go to Yummi breadcrumbs: Options page and switch there Installation to Manual</li>
<li>Place this code to your templates:
<pre><code><?php if(function_exists('yummi_breadcrumbs')) yummi_breadcrumbs(); ?></code></pre>
and this code to your AMP (Accelerated Mobile Pages) templates:
<pre><code><?php if(function_exists('yummi_breadcrumbs_amp')) yummi_breadcrumbs_amp(); ?></code></pre>
</li>
</ol>
